Technical Specifications: Standard Vertebrate Field Guide

Model: Morphological Identification Key, General Purpose Manufacturer: Systema Naturae (est. 1735) Current Edition: Continuously Updated Coverage: All vertebrate classes

Detection Parameters

| Parameter | Specification | |-----------|--------------| | Input channel | Visual morphology (external features, skeletal structure, coloration, size) | | Resolution | Phenotypic — resolves differences visible to trained human observer at arm's length | | Sensitivity | High within resolution band; zero below it | | False negative rate | ~50% (see Note 1) | | False positive rate | Not applicable — instrument does not generate false positives; it generates false unities |

Note 1: Recent meta-analysis (Zhang et al., Proc. R. Soc. B, 2026; n = 373 studies) indicates that each morphologically-identified vertebrate species contains approximately two genetically distinct species. This ratio is consistent across all major vertebrate groups (mammals, birds, reptiles, amphibians, actinopterygian and chondrichthyan fishes). The false negative rate refers to species present in the sample but not resolved by the instrument.

Operating Characteristics

• Produces definitive output for all inputs within resolution band.

• Output format: species name (Latin binomial), range description, habitat notes, illustration.

• Output does not include confidence intervals, resolution disclaimers, or uncertainty flags.

• User experience: answer. Not: hypothesis.

Known Limitations

1. Cannot resolve cryptic species (genetically distinct, morphologically identical organisms). 2. Cannot detect genetic divergence below phenotypic threshold. 3. Does not report when a classification is at or near resolution boundary. 4. Produces complete-appearing output regardless of actual completeness.

Note: Limitations 3 and 4 are architectural, not correctable by firmware update. The instrument's design precludes self-reporting of resolution boundaries. Output will always appear complete. Users seeking sub-resolution diversity should employ supplementary instruments (molecular sequencing, nuclear and mitochondrial DNA analysis).

Warranty

This instrument has been in continuous operation since 1735. It has been used to classify approximately 70,000 vertebrate species. Approximately 70,000 additional species may exist below its resolution.

The manufacturer does not guarantee that the number of species reported corresponds to the number of species present.

The manufacturer does not guarantee that the user will notice.
